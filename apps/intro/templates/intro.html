Grimwire 0.1.0

A Web OS with Users in Mind.

Structured

Grimwire uses Web Workers and RESTful messaging to form a program architecture in the browser. Links, forms, and Ajax calls target in-browser Javascript with "HTTP Local."

Learn More »

User-Driven

Grimwire's fully client-side toolset allows applications to be added, modified, and reconfigured without reloading the page. Take control of privacy and usability without owning the host.

Learn More »

Multi-Threaded JS

Grimwire's applications use as many Workers as they need. Take advantage of the latest HTML5 APIs without ever writing client-side code.
